# K72 Agency Clone

A modern frontend clone of the **K72 Agency** website built using **React**, **Vite**, **Tailwind CSS**, and **GSAP**. This project recreates the landing page and animations of the original website as a frontend practice project.

> ⚠️ This project is created for educational and portfolio purposes only. All design credits belong to K72 Agency.

---

## 🚀 Tech Stack

- React.js
- Vite
- Tailwind CSS
- GSAP
- React Router DOM

---

## ✨ Features

- Responsive landing page
- Hero section with embedded video
- Animated SVG stroke around the "CREATIVE" text
- GSAP-powered animations
- Scroll-based interactions using ScrollTrigger
- Custom typography
- Agency and Projects pages
- Modern UI inspired by K72

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/k72-clone.git
```

Navigate into the project

```bash
cd k72-clone
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open

```
http://localhost:5173
```
